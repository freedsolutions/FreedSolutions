Privacy Policy

Private LinkedIn API application for personal use only.

This Privacy Policy applies to a private LinkedIn API application operated by Adam Freed for individual use only. The application is not shared with the public, and Adam Freed is the sole user.

Last updated: March 14, 2026

1. Data Collection

This application may access limited LinkedIn account information that is explicitly authorized through LinkedIn's API, including profile name and email address.

The application is intended only for Adam Freed's own account and personal use. It is not designed to collect or process data from other LinkedIn members.

2. Data Usage

LinkedIn data accessed by this application is used solely for personal development, testing, debugging, and evaluation of application functionality.

No LinkedIn data is stored permanently. Any temporary data, tokens, or session information used during authentication or testing is retained only as long as necessary for the immediate operation of the application and is then deleted or allowed to expire.

LinkedIn data is not used for advertising, marketing, profiling, resale, or any commercial redistribution.

3. Third-Party Sharing

No LinkedIn data accessed by this application is shared with, disclosed to, or sold to third parties.

Because this application is for private individual use only, no clients, customers, public users, or third-party users are given access to LinkedIn data through the application.
