FreedSolutions Workflow Quickstart

1) Feature Prep (Browser Claude UI)
FEATURE_PREP: Draft/update FEATURE_CARD.md to implementation-ready quality (goal, scope, out-of-scope, constraints, acceptance, risks). Return final markdown only.

2) Claude Code Implement
CLAUDE_PHASE: Use agents/claude-feature-implementer.md with FEATURE_CARD.md.

3) Codex Review/Patch
CODEX_PHASE: Use agents/codex-commit-review-patcher.md to review and patch commit <hash>, then finalize SMOKE_TEST.md.

4) Browser Smoke
SMOKE: In browser Claude extension, paste the full contents of SMOKE_TEST.md into a new thread and run smoke tests. DO NOT CLOSE OUT THE BROWSER.

Required gates:
- End of step 2: HANDOFF_TO_CODEX
- End of step 3: DO NOT PUSH YET - Awaiting browser smoke RESULT
- Push only after step 4 returns: RESULT: PASS
