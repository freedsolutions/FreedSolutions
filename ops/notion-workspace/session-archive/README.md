# Session Archive

Archived snapshots of `ops/notion-workspace/session-active.md` belong in this directory.

## Convention

- Copy the prior `session-active.md` into this folder at the end of a session when a durable snapshot is useful.
- Use a dated filename such as `2026-03-20-session-62.md`.
- Keep the active handoff concise; move longer historical context into archived snapshots instead of letting `session-active.md` grow without bound.
- The Notion `Session - Archive` page can remain as legacy historical context, but the repo archive is the canonical source for future session snapshots.
